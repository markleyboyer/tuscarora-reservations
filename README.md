# The Tuscarora Club - Reservation System

A fully functional web-based reservation system for The Tuscarora Club, featuring member authentication, calendar views, and booking management.

## Features

- **Member Login System** - Unique username authentication
- **Interactive Calendar** - Month and week views
- **Smart Booking Flow** - Room selection, guest details, and meal planning
- **Lazy Lodge Priority System** - Provisional bookings for repeat users
- **Multi-Room Warnings** - Automatic alerts for large bookings
- **Admin Dashboard** - Room inventory and system configuration
- **Reservation Management** - View and cancel bookings

## Demo Accounts

- `member1`, `member2`, `member3` - Regular members
- `admin` - Administrator with full access

## Key Business Rules

1. **Lazy Lodge Priority**: Members can rent Lazy Lodge once per year with priority. Subsequent bookings are "provisional" and can be bumped by members who haven't used it yet.

2. **Multi-Room Threshold**: Booking 5+ rooms simultaneously triggers a House Committee Chairman consultation prompt.

3. **Meal Planning**: Breakfast, Lunch, and Bar Supper with packed meal options.

## Properties

- **Farm House**: 6 rooms (3 with bathroom, 3 shared)
- **Club House**: 6 rooms (3 with bathroom, 3 shared)
- **Lazy Lodge**: 2 rooms (both with private bathrooms)

## Technology Stack

- React 18
- Tailwind CSS
- Lucide Icons
- Pure JavaScript (no build process required)

## Local Development

Simply open `index.html` in a web browser. No build process or server required!

## GitHub Pages Deployment

This site is designed to work seamlessly with GitHub Pages. Just push to your repository and enable GitHub Pages in the repository settings.
